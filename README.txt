 Weather App

This is a simple weather application that fetches real-time weather data using the OpenWeather API. You can search for a city and get the temperature and weather forecast.

 How to Run Locally

 Step 1: Get Your OpenWeather API Key
To use this app, you need to get your own API key from OpenWeather:

1. Go to [OpenWeather](https://openweathermap.org/).
2. Sign up for a free account.
3. After logging in, go to the API keys section and generate your API key.

 Step 2: Set Up Your API Key

1. Download or clone this repository.
2. Open the `app.js` file.
3. Replace the placeholder `'YOUR_OPENWEATHER_API_KEY'` with your actual API key from OpenWeather:

   ```javascript
   const apiKey = 'YOUR_OPENWEATHER_API_KEY'; // Replace with your own key
